About SafeRoute

Traditional navigation apps optimize strictly for speed and distance. SafeRoute changes the paradigm by utilizing Machine Learning to prioritize pedestrian safety, routing users around historically high-crime zones and providing live guardian monitoring.

AI Pathfinding

SafeRoute utilizes K-Means Clustering on historical crime datasets to generate risk zones. A custom heuristic algorithm powered by OSMnx calculates safe street weights, applying penalties to road nodes near violent crime clusters.

Dataset & Mapping

Built using massive public safety datasets. The dynamic street graph is rendered flawlessly using Mapbox GL JS, ensuring a smooth, dark-mode focused UI tailored for nighttime navigation.

Guardian Mode

Once a route is active, Guardian Mode acts as a digital chaperone. If route deviations or suspicious prolonged stops are detected, the app automatically triggers an SOS alert and shares live GPS coordinates via WhatsApp.
